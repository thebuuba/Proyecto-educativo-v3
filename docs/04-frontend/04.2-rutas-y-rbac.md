# Rutas y Control de Acceso (RBAC)

| Ruta | Etiqueta | Ícono | Roles permitidos | Sidebar |
|------|----------|-------|------------------|---------|
| `/` | Inicio | LayoutDashboard | Todos | Sí |

| `/asignaturas` | Asignaturas | BookOpen | admin, director, coordinator, teacher | No |
| `/cursos` | Cursos | LibraryBig | admin, director, coordinator | Sí |
| `/horario` | Horario | CalendarClock | admin, director, coordinator, teacher | No |
| `/asistencia` | Asistencia | CalendarCheck | admin, coordinator, teacher | Sí |
| `/calificaciones` | Evaluación | GraduationCap | Todos | Sí |
| `/planificaciones` | Planificación | NotebookPen | admin, director, coordinator, teacher | Sí |
| `/matriz` | Matriz | Grid3x3 | admin, director, coordinator, teacher | No |
| `/reportes` | Reportes | ChartNoAxesCombined | admin, director, coordinator, teacher | Sí |
| `/configuracion` | Administración escolar | Settings | admin | Sí |
| `/perfil` | Perfil | UsersRound | Todos | No |

## Control de acceso

### RequireAuth
Componente que envuelve rutas protegidas en `App.tsx`:
1. Si loading → muestra "Cargando sesión..."
2. Si no autenticado → redirige a `/login`
3. Si no tiene rol → redirige a `/sin-acceso`
4. Si autorizado → renderiza children

### Roles disponibles
`'admin' | 'director' | 'coordinator' | 'teacher' | 'student' | 'guardian' | 'viewer'`

### Sidebar
El `Sidebar` filita los items según `navigationRoutes.filter(item => hasRole(item.allowedRoles))`, mostrando solo las rutas que el usuario puede ver y que tienen `showInSidebar: true`.

## Lazy loading
