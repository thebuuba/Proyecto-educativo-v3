# Componentes UI

18 componentes reutilizables en `components/ui/`, todos exportados desde `index.ts`. Construidos con Tailwind CSS v4 y Lucide React para íconos.

| Componente | Props principales | Descripción |
|------------|------------------|-------------|
| **Button** | `variant: primary/secondary/ghost/outline/destructive`, `size: sm/md/icon`, `loading?: boolean` | Botón con soporte de spinner, animación en active |
| **Input** | `InputHTMLAttributes` | Input de texto con bordes redondeados y focus ring |
| **Select** | `SelectHTMLAttributes` | Select nativo con estilo consistente |
| **Textarea** | `TextareaHTMLAttributes` | Área de texto estilizada |
| **PasswordInput** | `label?: string`, más `InputHTMLAttributes` | Input con toggle mostrar/ocultar (Eye/EyeOff) |
| **Badge** | `tone: default/accent/success/warning/destructive/muted` | Etiqueta inline con anillo de color |
| **Card** | `children` | Contenedor con borde redondeado y sombra. Subcomponentes: `CardHeader`, `CardTitle`, `CardDescription`, `CardContent` |
| **Table** | `children` | Tabla simple. Subcomponentes: `TableHead`, `TableBody`, `TableHeaderCell`, `TableCell` |
| **Modal** | `title, description?, children, onClose` | Modal centrado con overlay, usa `useFocusTrap` |
| **Drawer** | `title, eyebrow?, children, onClose` | Panel lateral derecho (slide-in, max-w-xl) |
| **ConfirmDialog** | `title, description, confirmLabel?, cancelLabel?, destructive?, onConfirm, onClose` | Modal de confirmación con soporte async |
| **EmptyState** | `title, description, action?` | Estado vacío centrado con acción opcional |
| **ErrorState** | `message` | Barra de error con icono AlertCircle |
| **ErrorFallback** | `children` | ErrorBoundary (class component) con UI de recuperación |
| **LoadingState** | `message?` (defecto "Cargando...") | Texto de carga centrado |
| **PageShell** | `title, description, children?, actions?` | Layout de página estándar con header + contenido o EmptyState |
| **PageHeader** | `title, description, actions?` | Título + descripción + acciones opcionales |
