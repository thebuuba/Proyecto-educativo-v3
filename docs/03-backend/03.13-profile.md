# Módulo Profile

**Responsabilidad:** Perfil personal del usuario autenticado.

| Método | Ruta | Auth | Descripción |
|--------|------|------|-------------|
| GET | `/profile` | JWT | Perfil propio (id, email, fullName, avatarUrl, phone) |

Módulo mínimo: solo lectura del perfil del usuario autenticado. La actualización se maneja a través del módulo Users (`PATCH /users/:id`).
