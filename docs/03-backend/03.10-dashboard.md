# Módulo Dashboard

**Responsabilidad:** Panel principal con estadísticas agregadas y gestión de tareas.

| Método | Ruta | Roles | Descripción |
|--------|------|-------|-------------|
| GET | `/dashboard/stats` | Autenticado | Conteos agregados: estudiantes, docentes, matrículas |
| GET | `/dashboard/tasks` | Autenticado | Últimas 10 tareas |
| POST | `/dashboard/tasks` | admin, director, coordinator, teacher | Crear tarea |
| PATCH | `/dashboard/tasks/:id` | admin, director, coordinator, teacher | Actualizar tarea |
