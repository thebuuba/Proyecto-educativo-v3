# Módulo Courses

**Responsabilidad:** Configuración académica — gestión de grados, secciones, asignaturas y asignación materia-sección.

| Método | Ruta | Roles | Descripción |
|--------|------|-------|-------------|
| GET | `/courses/course-data` | Autenticado | Datos completos de cursos (grados, secciones, asignaciones, catálogos, año actual) |
| GET | `/courses/academic-levels` | Autenticado | Niveles académicos RD |
| GET | `/courses/cycles` | Autenticado | Ciclos académicos RD |
| GET | `/courses/modalities` | Autenticado | Modalidades RD |
| GET | `/courses/teachers` | Autenticado | Docentes activos |
| GET | `/courses/grades` | Autenticado | Listar grados |
| POST | `/courses/grades` | admin, director, coordinator | Crear grado |
| PATCH | `/courses/grades/:id` | admin, director, coordinator | Actualizar grado |
| DELETE | `/courses/grades/:id` | admin, director, coordinator | Desactivar grado (soft-delete) |
| GET | `/courses/grades/:gradeId/sections` | Autenticado | Secciones por grado |
| POST | `/courses/sections` | admin, director, coordinator | Crear sección |
| PATCH | `/courses/sections/:id` | admin, director, coordinator | Actualizar sección |
| DELETE | `/courses/sections/:id` | admin, director, coordinator | Desactivar sección |
| GET | `/courses/subjects` | Autenticado | Listar asignaturas |
| POST | `/courses/subjects` | admin, director, coordinator | Crear asignatura |
| POST | `/courses/assign-subject` | admin, director, coordinator | Asignar materia a sección (crea SectionSubject) |
| GET | `/courses/section-subjects` | Autenticado | Asignaciones activas |
| DELETE | `/courses/section-subjects/:id` | admin, director, coordinator | Desactivar asignación |

### Modelo SectionSubject
Vincula: schoolYear + grade + section + subject + teacher. Es la unidad base para horarios, calificaciones y asistencia por clase.
