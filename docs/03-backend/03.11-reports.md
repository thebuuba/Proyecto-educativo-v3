# Módulo Reports

**Responsabilidad:** Generación de reportes de estudiantes y exportación de datos.

| Método | Ruta | Auth | Descripción |
|--------|------|------|-------------|
| GET | `/reports/student/:studentId` | JWT | Todas las calificaciones de un estudiante |
| POST | `/reports/export` | JWT | Exportar calificaciones como JSON o CSV |

### Exportación

`POST /reports/export` acepta:
- `type`: `'student'` (reporte individual) o `'grades'` (calificaciones masivas)
- Parámetros adicionales según el tipo
- Responde con el archivo exportado
