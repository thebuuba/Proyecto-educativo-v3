# Módulo Users

**Responsabilidad:** Gestión de usuarios del sistema — listar, obtener, actualizar, consultar roles y permisos.

| Método | Ruta | Roles | Descripción |
|--------|------|-------|-------------|
| GET | `/users` | admin, director | Listar usuarios de la escuela |
| GET | `/users/permissions` | Autenticado | Obtener permisos por roleIds |
| GET | `/users/:id` | admin, director | Detalle de un usuario |
| PATCH | `/users/:id` | admin, director | Actualizar usuario (fullName, email, avatarUrl) |
| GET | `/users/:id/roles` | admin, director | Roles del usuario |
