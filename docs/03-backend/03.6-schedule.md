# Módulo Schedule

**Responsabilidad:** Gestión del horario escolar — bloques horarios y entradas de horario.

| Método | Ruta | Roles | Descripción |
|--------|------|-------|-------------|
| GET | `/schedule` | Autenticado | Listar horarios (filtros: sectionId, schoolYearId, teacherId, gradeId) |
| GET | `/schedule/sections` | Autenticado | Secciones con nombre de grado |
| GET | `/schedule/teachers` | Autenticado | Docentes activos |
| GET | `/schedule/subjects` | Autenticado | Asignaturas activas |
| GET | `/schedule/section-subjects` | Autenticado | Materias asignadas a secciones |
| GET | `/schedule/time-slots` | Autenticado | Listar bloques horarios |
| POST | `/schedule/time-slots` | admin, director, coordinator | Crear bloque horario |
| PATCH | `/schedule/time-slots/:id` | admin, director, coordinator | Actualizar bloque |
| DELETE | `/schedule/time-slots/:id` | admin, director, coordinator | Eliminar bloque (cascade entradas) |
| GET | `/schedule/entries` | Autenticado | Entradas con filtro dayOfWeek |
| POST | `/schedule/entries` | admin, director, coordinator | Crear entrada de horario |
| PATCH | `/schedule/entries/:id` | admin, director, coordinator | Actualizar entrada |
| DELETE | `/schedule/entries/:id` | admin, director, coordinator | Eliminar entrada |

### Modelos

- **TimeSlot:** Bloque horario (nombre, startTime, endTime, sequence) — único por escuela+secuencia
- **ScheduleEntry:** Asignación materia-sección en día+bloque (sectionSubjectId, timeSlotId, dayOfWeek, room)
