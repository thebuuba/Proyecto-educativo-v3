# Módulo Settings

**Responsabilidad:** Configuración de la escuela (perfil y años escolares).

| Método | Ruta | Roles | Descripción |
|--------|------|-------|-------------|
| GET | `/settings/school` | Autenticado | Datos de la escuela |
| PATCH | `/settings/school` | admin, director | Actualizar escuela (nombre, logo, datos regionales) |
| GET | `/settings/school-years` | Autenticado | Años escolares |
| POST | `/settings/school-years` | admin, director, coordinator | Crear año escolar |
| PATCH | `/settings/school-years/:id` | admin, director, coordinator | Actualizar año escolar |
| POST | `/settings/school-years/:id/set-current` | admin, director, coordinator | Establecer como año actual (transacción: desmarca otros) |
| GET | `/settings/academic-periods` | Autenticado | Periodos académicos |

### School data (MINERD)
El perfil de escuela incluye campos del sistema educativo dominicano: sector (público/privado), código regional, nombre regional, código de distrito, nombre de distrito, código de centro, turno, modalidad primaria, subsistemas habilitados.
