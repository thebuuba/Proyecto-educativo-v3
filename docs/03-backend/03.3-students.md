# Módulo Students

**Responsabilidad:** CRUD de estudiantes, gestión de matrículas, importación masiva, gestión de tutores.

| Método | Ruta | Roles | Descripción |
|--------|------|-------|-------------|
| GET | `/students` | Autenticado | Listar estudiantes con búsqueda/status/paginación |
| POST | `/students` | admin, director, coordinator | Crear estudiante |
| POST | `/students/import` | admin, director, coordinator | Importación masiva CSV |
| GET | `/students/grades-with-sections` | Autenticado | Grados+secciones para dropdowns |
| GET | `/students/:id` | Autenticado | Detalle de estudiante |
| PATCH | `/students/:id` | admin, director, coordinator | Actualizar estudiante |
| PATCH | `/students/:id/deactivate` | admin, director, coordinator | Desactivar (soft-delete) |
| GET | `/students/:id/enrollments` | Autenticado | Matrículas del estudiante |
| POST | `/students/enrollments` | admin, director, coordinator | Crear matrícula |
| DELETE | `/students/enrollments/:id` | admin, director, coordinator | Eliminar matrícula (cascade asistencias/notas) |
| GET | `/students/:id/guardians` | Autenticado | Tutores del estudiante |
| POST | `/students/:id/notify-guardians` | admin, director, coordinator, teacher | Notificar a tutores (stub) |

### DTOs destacados

**CreateStudentDto:** studentCode, firstName, lastName, documentId?, birthDate, gender?, address?
**CreateEnrollmentDto:** studentId, gradeId, sectionId, schoolYearId, enrollmentDate?, academicStatus?, isRepeating?
