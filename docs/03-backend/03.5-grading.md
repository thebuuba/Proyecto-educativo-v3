# Módulo Grading

**Responsabilidad:** Gestión de calificaciones y recuperación pedagógica.

| Método | Ruta | Roles | Descripción |
|--------|------|-------|-------------|
| GET | `/grading` | Autenticado | Listar calificaciones (filtro sectionSubjectId + academicPeriodId) |
| GET | `/grading/section-subjects` | Autenticado | Materias-sección para dropdowns |
| GET | `/grading/academic-periods` | Autenticado | Periodos académicos activos |
| GET | `/grading/students` | Autenticado | Estudiantes con notas existentes para grid |
| POST | `/grading/save` | admin, director, coordinator, teacher | Crear o actualizar calificación |
| GET | `/grading/student/:studentId` | Autenticado | Todas las calificaciones de un estudiante |

### Estados de calificación
- `draft` — Borrador, editable por el docente
- `published` — Publicada, requiere permisos adicionales para modificar
- `voided` — Anulada

### Modelo GradesRecord
Cada registro incluye: enrollmentId, sectionSubjectId, academicPeriodId, assessmentName, score, maxScore (defecto 100), weight (defecto 1), status.
