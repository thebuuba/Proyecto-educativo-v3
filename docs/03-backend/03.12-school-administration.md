# Módulo School Administration

**Responsabilidad:** Configuración de la escuela (perfil y años escolares).

| Método | Ruta | Roles | Descripción |
|--------|------|-------|-------------|
| GET | `/school-administration/school` | Autenticado | Datos de la escuela |
| PATCH | `/school-administration/school` | admin, director | Actualizar escuela (nombre, logo y datos generales) |
| GET | `/school-administration/school-years` | Autenticado | Años escolares |
| POST | `/school-administration/school-years` | admin, director, coordinator | Crear año escolar |
| PATCH | `/school-administration/school-years/:id` | admin, director, coordinator | Actualizar año escolar |
| POST | `/school-administration/school-years/:id/set-current` | admin, director, coordinator | Establecer como año actual (transacción: desmarca otros) |
| GET | `/school-administration/academic-periods` | Autenticado | Periodos académicos |

### School data (MINERD)
El perfil de escuela incluye datos generales: sector (público/privado), código de centro, turno, modalidad primaria y subsistemas habilitados.
