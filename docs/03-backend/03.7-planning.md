# Módulo Planning

**Responsabilidad:** Planificación didáctica — periodos académicos y entradas de planificación.

| Método | Ruta | Roles | Descripción |
|--------|------|-------|-------------|
| GET | `/planning` | Autenticado | Listar planificaciones (filtro sectionSubjectId) |
| GET | `/planning/academic-periods` | Autenticado | Periodos académicos (filtro schoolYearId) |
| POST | `/planning/academic-periods` | admin, director, coordinator | Crear periodo académico |
| PATCH | `/planning/academic-periods/:id` | admin, director, coordinator | Actualizar periodo |
| DELETE | `/planning/academic-periods/:id` | admin, director, coordinator | Eliminar periodo (cascade entradas+notas) |
| GET | `/planning/competencies` | Autenticado | Competencias fundamentales (filtro subjectId) |
| GET | `/planning/section-subjects` | Autenticado | Materias-sección (filtro teacherId) |
| GET | `/planning/entries` | Autenticado | Entradas de planificación |
| POST | `/planning/entries` | admin, director, coordinator, teacher | Crear entrada |
| PATCH | `/planning/entries/:id` | admin, director, coordinator, teacher | Actualizar entrada |
| DELETE | `/planning/entries/:id` | admin, director, coordinator, teacher | Eliminar entrada |

### Modelo PlanningEntry

Cada entrada incluye:
- **Competencia específica** e **indicador de logro**
- **Contenidos:** conceptuales, procedimentales, actitudinales
- **Estrategias** didácticas
- **Actividades:** inicio, desarrollo, cierre (JSON)
- **Recursos** y **método de evaluación**
- **Duración** en minutos y **fecha planificada**
- **Instrumentos de evaluación** y **evidencias**
