# Módulo Subjects

**Responsabilidad:** Listado simple de asignaturas dentro de una escuela.

| Método | Ruta | Auth | Descripción |
|--------|------|------|-------------|
| GET | `/subjects` | JWT | Listar asignaturas (ordenadas por nombre) |

Es el módulo más simple del backend — solo lectura pública para usuarios autenticados.
